Debugging & Pseudo Code

This assignment covers flowchart design and pseudocode debugging. Each flowchart represents a logical process, followed by the corrected pseudocode script.

Flowchart 1 — Miles Per Gallon

Debugged Pseudocode

start
  input milesTraveled
  input gallonsOfGasUsed
  milesPerGallon = milesTraveled / gallonsOfGasUsed
  output milesPerGallon
stop

Flowchart 2 — Cost Per Day

Debugged Pseudocode

start
  input rent
  costPerDay = rent / 30
  output costPerDay
stop

Flowchart 3 — Tablespoons to Teaspoons

Debugged Pseudocode

start
  input tablespoons
  teaspoons = tablespoons * 3
  output teaspoons
stop

Flowchart 4 — Monthly Budget

Debugged Pseudocode

start
  input monthlyPay
  input rent
  input utilities
  input groceries

  totalBills = rent + utilities + groceries
  discretionary = pay - totalBills

  output pay
  output totalBills
  output discretionary
stop
